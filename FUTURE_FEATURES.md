# VaultArc Future Features

## Native VaultArc Format (`.varc`)
- Per-file checksums in manifest
- Recovery metadata blocks
- Fast integrity scan mode
- Incremental/snapshot archive support

## Archive Resilience
- Partial extraction from damaged archives
- Repair-oriented workflows
- Corruption diagnostics with user guidance

## Security Expansion
- Encrypted secure-send bundles
- Secret scanning (API keys, `.env`, SSH keys)
- Policy presets for enterprise extraction controls

## Content Intelligence
- Duplicate detection before compression
- Archive diff mode
- Content classification and local AI summarization

## UX and Integration
- Explorer context menu integration:
  - Open with VaultArc
  - Extract Here
  - Extract to Folder
  - Compress to VaultArc archive
- File association registration wizard
- Improved preview system:
  - syntax highlighting
  - richer image preview
  - large file streaming

## Distribution
- Signed installer/MSIX channels
- Optional auto-update service
- Crash telemetry opt-in with privacy controls
